
在对齐配置文件后,可以使用三种方法进行导航,并统一评估.
```bash
export HF_HOME=/home/xzy/data/my_hf_home/hf_object_rel_nav
conda activate object-rel-nav
python main.py -c configs/defaults.yaml
python main.py -c configs/object_react.yaml
python main.py -c configs/tango.yaml

python scripts/evaluate_objecreact.py out/results/original
```

`resume_eval: True` 时,程序会检查 `out/results/<task>/<exp>/<split>/<max_start_distance>/` 下最新一次同方法同目标源的结果目录. 如果 `args.yaml` 和当前配置一致且还没有 `results_summary.csv`,则跳过已有 `success_status` 的 episode 并继续未完成部分;如果最新匹配结果已经完成,会新建结果目录并在终端提醒.
