
官方给出的配置文件有三个,分别对应三种控制器
```bash
configs
    defaults.yaml       # RoboHop
    object_react.yaml   # ObjectReact
    tango.yaml          # TANGO
    
    controller
        object_react_controller.yaml # ObjectReact 的配套参数
```

关键字段
```bash
start_idx
step_idx    # 任务场景隔几个取一次
end_idx
task_type  
max_start_distance
```
